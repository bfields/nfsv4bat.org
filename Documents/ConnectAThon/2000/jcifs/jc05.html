jCIFS so far...

jCIFS is still a small, volunteer effort. As is typical with Open Source projects, most of us have other things going on in our lives. Thus, coding time is scarce.

However, there is a CVS repository, and code is available.

  • We have some basic utility code.
  • There is SMB client code.
  • There is some name service code.
  • Preliminary CIFS documentation is on-line with more being developed.
