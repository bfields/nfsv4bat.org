We Guessed Right

The jCIFS project has gotten a lot of attention. Several companies have contacted us asking how they can participate in jCIFS.

Areas of interest include:

  • Using jCIFS to support a cifs: URI scheme, similar to nfs:.
  • Using jCIFS on palmtops, etc., to provide mobile access to "home base" services.
  • Using jCIFS in small, embedded servers (eg. CDRom servers).

This is exactly the kind of thing we envisioned.
