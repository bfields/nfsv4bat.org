What is jCIFS?

jCIFS is an Open Source development project with the goal of implementing the Common Internet File System (CIFS) in the Java language.

  • jCIFS code is being released under the GNU General Public License.
  • jCIFS is a "spin-off" of Samba, but is a separate project.
  • jCIFS is not a port of Samba to Java.

jCIFS was started as a result of architectural discussions between Andrew Tridgel and another Samba Team member. It became clear that two different goals were being discussed and that Samba should not be adapted to try to meet both. Something new was needed.
