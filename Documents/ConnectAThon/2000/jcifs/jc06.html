jCIFS Organization

Complete server or client packages may be generated, but that is not the overall goal.

jCIFS provides:

  • A forum for discussing issues, designs, and code.
  • A repository & clearning house for source code and documentation.
  • A contact point for interested parties, including the commercial sector.

The current CVS respository provides separate archives for each developer. The code is under GPL, so jCIFS developers freely exchange bits of code. There is also a directory for the agreed-upon "main" code set.
