What is jCIFS?

jCIFS is an Open Source implementation of the Common Internet File System (CIFS) in the Java language.

Kewl!

  • jCIFS is not a port of Samba to Java.
  • jCIFS is a "spin-off" of Samba, but it is a separate project.

jCIFS was started as a result of architectural discussions between Andrew Tridgel and another Samba Team member.
