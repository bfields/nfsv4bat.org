Evaluation and Selection of Base Platform

  • NFSv4Shell

    • TCL-based C program
      • Same goal as my project

  • Drawbacks

    • Not released to the public at project start
    • Less portable than Python, because it's written in C and uses rpcgen
      • Does not compile on Linux (rpcgen problem)
    • We have no experience with TCL
    • TCL lacks support for Object Oriented programming
