pynfs -Python NFSv4 tools

  • pynfs consists of:

    • rpc.py: RPC module
    • rpcgen.py: An RPC compiler
    • nfs4lib.py: NFSv4 library
    • nfs4client.py: Simple commandline client
    • nfs4st.py: Server test suite
    • epinfo2sxw.py: Extracts test case documentation from nfs4st.py
    • test_tree.py: Creates directory tree in preparation for running test suite
